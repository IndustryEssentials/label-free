## 常用快捷键

| 按键    | 功能         | 按键                                                      | 功能     |
| ------- | ------------ | --------------------------------------------------------- | -------- |
| ++q++   | 连续画框     | :material-mouse-move-vertical: 或者 ++plus++ / ++hyphen++ | 缩放图片 |
| ++w++   | 单个画框     | ++m++                                                     | 移动图片 |
| ++r++   | 四点画框     | ++f++                                                     | 隐藏类别 |
| ++s++   | 提交         | ++g++                                                     | 隐藏属性 |
| ++a++   | 切换上一张图 | ++h++                                                     | 隐藏选中 |
| ++d++   | 切换下一张图 | ++x++                                                     | 独显选中 |
| ++z++   | 撤销         | ++k++                                                     | 显示全部 |
| ++v++   | 恢复         | ++c++                                                     | 清空全部 |
| ++del++ | 删除         |                                                           |          |
