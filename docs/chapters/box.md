## 常用快捷键

| 按键        | 功能         |
| ----------- | ------------ |
| Q           | 连续画框     |
| W           | 单个画框     |
| R           | 四点画框     |
| S           | 提交         |
| A           | 切换上一张图 |
| D           | 切换下一张图 |
| Z           | 撤销         |
| V           | 恢复         |
| Delete      | 删除         |
| 滚轮；+ / - | 缩放图片     |
| M           | 移动图片     |
| F           | 隐藏类别     |
| G           | 隐藏属性     |
| H           | 隐藏选中     |
| X           | 独显选中     |
| K           | 显示全部     |
| C           | 清空全部     |
